## Как запустить?
сначала нужно запустить сервер и его базу
#### Перед запуском докера, запустите Docker Desctop, Docker CLI и т.п. - запустите ядро докера
```
cd server
docker-compose up -d
npm start
```

далее запускаем клиент
```
cd ..
cd client
npm run dev
```
